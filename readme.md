EduFlow - сервис визуализации школьного расписания на основе данных системы Nikasoft.

Проект предназначен для генерации актуального расписания в удобном графическом формате без необходимости установки клиентского приложения.

---

## Назначение проекта

EduFlow решает задачу наглядного и быстрого представления школьного расписания для учеников и педагогов, используя публичные данные расписания(NikaSoft) образовательных учереждений.

Проект ориентирован на использование в образовательной среде и может применяться как вспомогательный инструмент информирования.

---

## Основные возможности

- Генерация расписания в виде изображения
- Поддержка форматов данных Nikasoft
- Связка с My School
- Корректная обработка:
  - замен
  - подгрупп
  - кабинетов
  - учителей
- Отображение обрывов уроков с пометкой «безурочное время»
- Поддержка режимов отображения:
  - ученик
  - учитель
- Минималистичный и читаемый визуальный стиль

## Формат вывода

Результатом работы сервиса является изображение с расписанием, оптимизированное для просмотра на мобильных устройствах и десктопе.

Формат ориентирован на:
- компактность
- читаемость
- однозначное восприятие информации

---

## Конфиденциальность

EduFlow не осуществляет сбор, хранение или обработку персональных данных пользователей.

Используются только данные расписания, предоставляемые образовательными учреждениями в публичном виде.

---

## Статус проекта

Функциональность и формат вывода могут изменяться по мере развития проекта.
