# EduFlow

**Школьное расписание. Без боли.**

Современное Android-приложение, которое объединяет расписание NikaSoft, домашние задания и оценки из "Моя Школа" в одном удобном интерфейсе.

## Зачем это нужно

EduFlow объединяет лучшее из обоих миров: актуальное расписание + домашки и оценки, всё с современным интерфейсом в стиле Material You и тёмной темой.

Проект написан школьником - для школьников (и учителей тоже).

---

## Возможности

### Расписание (NikaSoft)
- Просмотр по дням с навигацией по неделям, автоопределение текущего дня
- Замены уроков — автоматически, с зачёркиванием отменённых
- Отображение кабинетов, преподавателей, подгрупп
- Режим **ученика** и **учителя**
- Безурочные промежутки отмечаются отдельно

### Домашние задания (Моя Школа / ФГИС)
- Просмотр заданий с группировкой «Ближайшие» / «Прошедшие»
- Скачивание прикреплённых файлов
- Открытие ЦДЗ прямо в приложении (библиотека МЭШ)
- Отметка выполнения одним тапом с синхронизацией на портал *(Alpha)*

### Оценки
- Группировка по предметам и триместрам
- Автоматический расчёт среднего балла
- Подсветка контрольных работ (вес оценки, тип КР, комментарии учителя)

### Дополнительно
- Тёмная и светлая тема
- Кэширование — быстрый запуск без лишних запросов *(Alpha)*
- Push-уведомления о новых домашках в 17:00 (OneSignal. в данный момент допиливаю)
- Праздничные анимации

## Конфиденциальность

EduFlow не собирает, не хранит и не передаёт персональные данные пользователей. Используются только публичные данные расписания образовательных учреждений и токен авторизации пользователя, хранящийся локально на устройстве.

## Правовая информация

EduFlow — независимый проект, не аффилированный с NikaSoft или ФГИС "Моя Школа". Все права на данные расписания принадлежат соответствующим образовательным учреждениям.
